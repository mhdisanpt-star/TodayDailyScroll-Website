Privacy Policy

Last updated: July 7, 2026

Today Daily Scroll ("TDS", "we") operates an Editorial Platform for video content. This policy describes what data we collect and how we use it, including integrations with YouTube, TikTok, and Google OAuth.

Data we collect

We collect data needed to run editorial and publishing workflows:

  • Account and channel identifiers from OAuth authorization
  • OAuth tokens for YouTube and TikTok API access
  • Video metadata, upload status, and analytics from platform APIs
  • Editorial workflow data: review decisions, job status, operational logs
  • Technical logs for reliability and security
  • Contact information when you email us

We do not sell user data.

How we use data

  • Run editorial production, review, and quality workflows
  • Upload and verify content on authorized platforms
  • Sync analytics for creator dashboards
  • Maintain system reliability and security
  • Respond to support and privacy requests

We use data only for editorial and publishing operations. We do not use it for advertising or resale.

OAuth permissions

When you connect YouTube, TikTok, or Google accounts, you grant specific OAuth scopes. We request the minimum scopes needed for the features you enable. You can revoke access through each platform's account settings at any time.

OAuth tokens are stored on the server side and are not exposed on public pages.

YouTube API

TDS uses YouTube API Services under the YouTube API Services Terms of Service. YouTube data is used for editorial workflow, upload, verification, and performance analytics within TDS.

TikTok API

Where TikTok Content Posting API is enabled, TDS uses access tokens to upload and verify content you authorize. TikTok data is not used outside the publishing features you configure.

Cookies

This public website is static and does not set tracking cookies. The TDS dashboard, when self-hosted, may use session cookies for authentication. We do not use advertising cookies.

Security

We use access controls and standard server-side practices to protect OAuth tokens, credentials, and operational data. Access to production systems is limited to authorized operators.

Retention

We retain data as long as needed for editorial operations, compliance, and troubleshooting. Audit logs may be kept for operational integrity. You can request deletion subject to legal requirements.

Your rights

You may request access, correction, or deletion by contacting us. Revoking OAuth access through Google, YouTube, or TikTok disables further API operations for that account.
